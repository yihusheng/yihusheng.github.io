# Music — 导航栏说明

**此页面不参与统一导航栏系统。**

## 原因
Music（音乐解锁）是一个独立的全屏 SPA 应用，有自己完整的 UI 和交互逻辑。引入导航栏会导致：
- 内容与导航按钮重叠
- 布局错乱
- 用户体验下降

## 规则
1. **不要**将此页面加入 `Tools/navbar.js` 的 `NAV_ITEMS` 数组
2. **不要**在此页面的 `<head>` 中引入 `navbar.css` 和 `navbar.js`
3. **不要**给 `<body>` 添加 `navbar-overlay` 类
4. 此页面完全独立运行，不依赖导航栏系统

## 首页入口
首页 `index.html` 底部的"✕"链接指向 `/Music`，这是唯一入口。
